# Media Scraper

## Tech Stack

### Back-end

- Node.js
- TypeScript
- SQLite
- Express
- Axios
- Cheerio

### Front-end

- ReactJS (Vite)
- TypeScript
- TanStack React Query
- Tailwind CSS
- Shadcn UI

## How to run

### Docker

```bash
docker-compose up --build
```

The Client will be running at `http://localhost:3000` and the Server will be running at `http://localhost:3001`.

### Local

```bash
# Install dependencies
npm install

# Run the server
cd server
npm run dev

# Run the client
cd client
npm run dev
```

The Client will be running at `http://localhost:5173` and the Server will be running at `http://localhost:3001`.
